# Klasyfikacja-wydolności-fizycznej-
Problem klasyfikacji wydolności osób, których celem jest przewidywanie klasy uwzględniając parametry fizyczne tj.

Wiek (age), Płeć (gender) Wzrost (height_cm) Waga (weight_kg) Procent tkanki tłuszczowej (body fat_%) Parametry siłowe i wytrzymałościowe (np. ciśnienie, siła chwytu, liczba podciągnięć, długość skoku w dal itp.) Klasa wydolności (class) – etykieta do przewidywania (A, B, C, D), klasa A najwyższa

Testowane są różne klasyfikatory

Automatyczny wybór najlepszego modelu na podstawie wyniku recall.
Predykcja na nowych danych

Po znalezieniu najlepszego modelu można wprowadzać nowe dane (np. cechy nowej osoby) i klasyfikować jej wydolność fizyczna.

Klasyfikacja poziomu sprawności fizycznej na podstawie pomiarów
ma możliwości zastosowanie w sporcie, rehabilitacji, diagnostyce zdrowia i treningach personalnych
